# 🐍 Snake Game

Простая игра **«Змейка»**, написанная с использованием **HTML**, **CSS** и **JavaScript**.

## 📋 Описание

Игрок управляет змейкой с помощью клавиш-стрелок. Цель игры — собирать еду, увеличивая длину змейки и количество очков. Игра заканчивается, если змейка столкнётся со стеной или с собственным телом.

## 📂 Структура проекта

```
snake_site/
│
├── index.html      # Главная страница
├── style.css       # Стили оформления
├── script.js       # Логика игры
└── README.md       # Документация проекта
```

## 🚀 Запуск проекта

1. Скачайте или клонируйте проект.
2. Откройте папку проекта в Visual Studio Code.
3. Запустите файл `index.html` через браузер или расширение **Live Server**.

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| ⬆️ | Движение вверх |
| ⬇️ | Движение вниз |
| ⬅️ | Движение влево |
| ➡️ | Движение вправо |

## ⚙️ Используемые технологии

- HTML5
- CSS3
- JavaScript (ES6)
- Canvas API

## 🕹️ Правила игры

- Змейка начинает движение с центра игрового поля.
- Красный квадрат — это еда.
- После съедения еды змейка увеличивается на один сегмент.
- За каждую съеденную еду начисляется 1 очко.
- Столкновение со стеной или собственным телом завершает игру.

## 📸 Интерфейс

На странице отображаются:

- заголовок игры;
- игровое поле (Canvas);
- счёт игрока.
